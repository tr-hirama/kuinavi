# ハウス杭ナビ変換 — 使用マニュアル

杭ナビ CSV（mm 単位）を GL.csv（m 単位）に変換する Web ツールです。
全処理はブラウザ内で完結し、ファイルが外部にアップロードされることはありません。

公開URL: https://tr-hirama.github.io/kuinavi/

---

## 1. ファイルを読み込む

以下のいずれかで CSV を読み込めます。

- **ドラッグ&ドロップ**: 画面左上の点線エリアに CSV をドロップ
- **ファイル選択**: ツールバーの「ファイル選択」ボタンから選ぶ

### 入力 CSV の形式

```
ポイント名 , Y座標(mm) , X座標(mm) , Z座標(mm)
```

- Z 座標は空欄でも可（0 として扱う）
- ポイント名の頭文字で種別を判定:
  - `P` … 杭（編集対象 / 色分け表示）
  - `BM` … 水準点（濃紺ダイヤモンド / Z 編集対象）
  - `K` … 基準点（赤い三角＋黒線で結合）
  - `H` … 境界（緑の四角＋折れ線）
  - `S` … 機器位置（配置図には非表示）

### 変換式

| 出力 | 計算式 |
|---|---|
| X (m) | 入力 X (mm) / 1000 + 10.0 |
| Y (m) | 入力 Y (mm) / 1000 + 10.0 |
| Z (m) | 入力 Z (mm) / 1000 |

---

## 2. 画面構成

```
┌─────────────────────┬────────────────────────────┐
│ データグリッド          │ 配置図 / 出力CSV / 生データ │
│ (全点)                  │ (タブ切替)                  │
└─────────────────────┴────────────────────────────┘
│  ステータスバー (操作結果)                          │
```

- **左**: 全点の入力 / 出力座標を一覧表示（P 杭の Z セルは直接編集可能）
- **右**: 配置図がデフォルト、タブで出力プレビューと元データに切替
- **下**: 直前の操作結果や読み込み件数を表示

---

## 3. 配置図の操作

| 操作 | 動作 |
|---|---|
| マウスホイール | カーソル位置を中心にズームイン / アウト |
| ドラッグ | パン（掴んで動かす） |
| クリック | 最寄りの点を選択（P > K > H の優先順） |
| ダブルクリック | ズーム / パンを初期状態にリセット |

### 凡例の色
P 杭の Z 値を**本数の多い順**に色分け:

1. 水色（最多）
2. 薄緑
3. 朱色
4. 以降: 黄土・紫・ティール・ピンク・ブラウン

> 色は読込時の本数で決まり、Z 値編集後も基本的に保たれます（同じ delta で平行移動した場合は完全に同色）。

---

## 4. P 杭の Z 値を編集する

### A. グリッドで直接編集
グリッドの「入力 Z」セルを**ダブルクリック** → 値を入力 → Enter で確定。
P 杭以外は編集できません。

### B. 「選択 P の Z 編集」ボタン
グリッドや配置図で P を選択した状態で押すと、mm 単位の入力ダイアログが開きます。

### C. 「グループ変更」ボタン
同じ Z 値を持つ P 群だけを変更します。

1. 対象の Z グループをドロップダウンで選択
2. モード:
   - **新しい Z 値に設定** (Z = 値)
   - **加算する** (Z = Z + Δ)
3. 値を入力 → OK

### D. 「全本変更」ボタン
全 P 杭を対象とする一括操作。4 モード:

| モード | 動作 |
|---|---|
| 一律設定 | 全 P の Z を指定値に置き換え |
| 全 P に加算 | 全 P に Δ を加算 |
| S 点 Z 減算 | 全 P から S 点の Z を引く（基準合わせ） |
| グループ基準シフト | 指定グループが目標値になるよう差分を計算 → 全 P に適用 |

> 全本変更では色は変わりません（点の意味的グループが保たれるため）。

---

## 5. 座標変換（選択 P 基準で全点シフト）

選択した P 杭を基準点として、X / Y / Z の差分を**全点 (P / K / H / S)** に平行移動します。

### 手順
1. P 杭を 1 つ選択
2. ツールバーの「座標変換」ボタン
3. ダイアログに現在の X / Y / Z (m) が表示
4. 目標座標を入力 → 差分プレビュー表示
5. OK で全点に適用

### 使用例
- 「P1 を実測したら (10.500, 12.300, 5.200) m だった」
  → ダイアログに入力 → 全点が同じだけシフト
- KBM 変更、座標系オフセットの一括調整

---

## 6. 結果を保存する

1. ツールバーの「**GL.csv を保存**」ボタンを押す
2. 初回: ネイティブ保存ダイアログで**任意のフォルダ**を選択
3. 2 回目以降: 同じ場所に上書き（ダイアログなし）
4. クリアまたは新規読込で保存先はリセット

### エンコーディング
ツールバーで選択:

| 値 | 用途 |
|---|---|
| Shift-JIS（デフォルト） | 既存ツールとの互換 |
| UTF-8 (BOM 付) | Excel で文字化けしない |

### 出力 CSV の形式
```
1,10.500,10.300,5.000,
2,11.500,10.300,5.000,
K1,12.000,11.000,0.000,
H1,13.000,11.500,0.000,
S,11.000,11.000,0.000,
```

| 項目 | 仕様 |
|---|---|
| 対象 | **全点**（P / K / H / S すべて） |
| P 名前 | 先頭の `P` を除去（`P1` → `1`） |
| K / H / S | そのまま |
| 並び順 | P を番号昇順で先頭 → 非 P は入力順 |
| 行末 | カンマ `,` で終わる |

### 出力プレビュー
「出力 CSV」タブで保存前に内容を確認できます。
「クリップボードにコピー」ボタンで他アプリに直接貼り付け可能。

---

## 7. 取り消し / やり直し (Undo / Redo)

| 操作 | ショートカット |
|---|---|
| 元に戻す | `Ctrl+Z` / ツールバーの ↶ ボタン |
| やり直し | `Ctrl+Y` または `Ctrl+Shift+Z` / ツールバーの ↷ ボタン |

対象となる操作:
- Z 値編集（グリッド直接 / 選択 P / グループ / 全本）
- 座標変換
- クリア

最大 **50 件** まで履歴を保持します。新規 CSV 読込で履歴はリセットされます。
テキスト編集中（セル編集中など）はネイティブの編集 Undo が優先されます。

## 8. その他の機能

- **クリア**: 読み込んだデータを全て削除（Ctrl+Z で復元可能）
- 編集された Z セルはグリッド上で**緑色に強調**
- ブラウザは Chrome / Edge を推奨（ネイティブ保存ダイアログが使えます）
- Firefox / Safari は Downloads フォルダへの自動ダウンロードでフォールバック

---

## 典型ワークフロー

1. 杭ナビから CSV を出力 → ブラウザにドロップ
2. 配置図で全体を確認（必要に応じてズーム）
3. グリッドで Z 値を編集 or 一括変更で調整
4. 必要なら「座標変換」で基準点合わせ
5. 「GL.csv を保存」で出力 → 次工程へ
